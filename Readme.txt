
Ballet and Python

Скрипты Python:
+ Create folders.ipynb - скрипт для создания необходимых папок и установки нужных библиотек
+ Create sequences.ipynb - для расскадровки видеофайлов балетных номеров (скачанных с Youtube)
+ Resize frames.ipynd - для изменения и унификации размера кадров (jpeg-файлов)
+ Create videosamples.ipynb - для создания видеосэмплов для предпросмотра этих реализаций конкретных реализаций балетных элементов. 
+ Create ballet.ipynb - для создания балетного видео из списка кадров и прикрепления аудиодорожки

Конструктор балетных номеров:
+ Create_Ballet_v1.3.xlsx

Алгоритм создания Библиотеки балетных элементов
+ скачай скрипты, xlsx и папку FromYoutube_mp4 в одну папку
+ Запусти Create folders.ipynb: будут созданы необходимые
+ Запусти Create sequences.ipynb: для расскодровки видеофайлов
+ Запусти Resize frames.ipynd
+ Запусти  Create videosamples.ipynb
+ папку FromYoutube_jpeg можно удалить

Библиотека готова.
Создавай балетные номера в Create_Ballet_v1.3.xlsx и создавай балетные видео с помощью Create ballet.ipyn
